# quiz helper by tengocj

simple chrome extension that helps with mcq quiz pages.

## what it does

- reads the current question
- reads possible answers
- reads code blocks if there is one
- optional: uses your openai key to suggest answer(s)
- optional: marks suggested answers (green) and the others (red)

## video

demo video: [Video](./demo.mp4)

## quick start

1. open `chrome://extensions`
2. enable developer mode
3. click `load unpacked`
4. select this folder
5. open extension popup
6. click `Enable monitoring`
7. click `Open monitor window`

## openai (optional)

1. in monitor window paste your openai api key
2. enable `auto answer`
3. done
